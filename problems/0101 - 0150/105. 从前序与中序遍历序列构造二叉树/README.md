根据一棵树的前序遍历与中序遍历构造二叉树。

**注意:**

你可以假设树中没有重复的元素。

例如，给出

```
前序遍历 preorder = [3,9,20,15,7]
中序遍历 inorder = [9,3,15,20,7]
```

返回如下的二叉树：

```
    3
   / \
  9  20
    /  \
   15   7
```

**思路:**

根据 `preorder[0]` 创建根节点，之后根据 `inorder` 中 `preorder[0]` 的位置得到左子树和右子树的节点数量，进而得到左子树和右子树的前序遍历和中序遍历，递归地计算左子树和右子树，最终与根节点合并。
