给定一个正整数，返回它在 Excel 表中相对应的列名称。

例如，

```
    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB
    ...
```

**示例 1:**

```
输入: 1
输出: "A"
```

**示例 2:**

```
输入: 28
输出: "AB"
```

**示例 3:**

```
输入: 701
输出: "ZY"
```

**思路:**

将整数转换为 26 进制，由于没有表示 0 的字母，所以当余数为 0 时，使用 Z (即 26) 表示当前位，n 整除 26 后继续减一。
