给定一个字符串数组，将字母异位词组合在一起。字母异位词指字母相同，但排列不同的字符串。

**示例:**

```
输入: ["eat", "tea", "tan", "ate", "nat", "bat"]
输出:
[
  ["ate","eat","tea"],
  ["nat","tan"],
  ["bat"]
]
```

**说明:**

- 所有输入均为小写字母。
- 不考虑答案输出的顺序。

**思路:**

遍历数组，使用哈希表记录已经遍历的字符串。对于每一个字符串，将字符排序后得到一个新的字符串，如果哈希表中存在，表明其与之前的某个字符串是字母异位词，添加进结果数组中；如果不存在，新建一个结果数组并存入哈希表。
