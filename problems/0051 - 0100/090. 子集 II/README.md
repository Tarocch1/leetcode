给定一个可能包含重复元素的整数数组 ***nums***，返回该数组所有可能的子集（幂集）。

**说明:** 解集不能包含重复的子集。

**示例:**

```
输入: [1,2,2]
输出:
[
  [2],
  [1],
  [1,2,2],
  [2,2],
  [1,2],
  []
]
```

**思路:**

思路与[78. 子集](https://github.com/Tarocch1/leetcode/tree/master/problems/0051%20-%200100/078.%20%E5%AD%90%E9%9B%86)基本一致。由于数组中存在重复元素，因此在遍历时应跳过已经遍历过的数，为此，先将数组进行排序。
