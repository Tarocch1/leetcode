给定两个二叉树，编写一个函数来检验它们是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

**示例 1:**

```
输入:      1         1
          / \       / \
         2   3     2   3

        [1,2,3],  [1,2,3]

输出: true
```

**示例 2:**

```
输入:      1         1
          /           \
         2             2

        [1,2],     [1,null,2]

输出: false
```

**示例 3:**

```
输入:      1         1
          / \       / \
         2   1     1   2

        [1,2,1],  [1,1,2]

输出: false
```

**思路:**

采用递归的方法，首先比较两棵树的根节点，如果不同直接返回 false；如果相同，递归地比较两棵树的左子树和右子树，结果取与。
